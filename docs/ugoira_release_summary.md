# PixivVault 「うごイラ（アニメーションイラスト）」完全対応リリースまとめ

PixivVault が新たに **うごイラ（アニメーションイラスト：`illustType == 2`）** の自動ダウンロードと3種類の保存形式に完全対応いたしました！

従来は通常イラスト・漫画および小説のみが対象でしたが、本アップデートにより、動くイラスト作品もフレームごとの遅延時間（タイミング情報）を忠実に再現した状態で、お好みのフォーマットへ自動変換して保存できます。

---

## 🌟 主な機能とハイライト

### 1. 選べる3つのうごイラ保存形式
設定画面の **「4. その他の設定」 > 「うごイラの保存形式」** より、ご自身の用途やお好みに合わせて以下の3つの保存モードを自由にご選択いただけます。

| 保存形式 | 概要・特徴 | 推奨用途 |
| :--- | :--- | :--- |
| **GIF動画 (`gif`)** *(デフォルト)* | PixivからダウンロードしたZIPデータをメモリ上で高速展開し、各コマのフレーム遅延時間 (`delay`) を正確に適用したアニメーションGIFを自動生成します。 | 一般的な閲覧・SNS共有・ビューワー表示に最適 |
| **MP4動画 (`mp4`)** | OpenCV または imageio ライブラリを使い、高画質・高圧縮なMP4動画ファイルへ自動エンコードします。<br>※ライブラリが利用できない環境では、安全にGIF保存へ自動フォールバックします。 | 高画質でのコレクション、動画プレイヤーでの鑑賞・編集 |
| **すべてフォルダ格納する (`folder`)** | ZIP内の全コマ連番画像ファイル (`000000.jpg`, `000001.jpg` 等) と、各コマの表示ミリ秒時間を記録した `ugoira_meta.json` を展開フォルダへまるごと保存します。 | 動画編集ソフトや自作スクリプト等での2次加工・完全生データの保管 |

---

### 2. オンメモリ処理による高速変換と安心の排他制御
- **高速オンメモリ変換**: 巨大な一時ファイルをストレージ（SSD/HDD）に何度も書き込むことなく、PCのメインメモリ（RAM）上だけでZIP解凍とGIF/MP4構築を完結させるため、非常に高速に処理が終わります。
- **排他制御の徹底**: 既存の作者単位ロック機構 (`_user_locks`) と統合されており、GUI手動バックグラウンド実行・自動チェック・ブラウザ拡張機能連携キューなどが同時に同一作者をダウンロードしても、ファイルやアーカイブが破損することはありません。

---

### 3. あらゆるダウンロードモード＆差分チェックに対応
以下の全てのダウンロードフローにて、うごイラ作品が自動認識・保存されます。
- **個別DL**: ユーザーID指定による一括・差分ダウンロード
- **フォロー中一括DL**: フォローリストに登録された作者たちの一括バックアップ
- **ブックマークDL**: ブックマーク作品のフラット配置（direct）および階層配置（link）モード
- **差分スキップ検知**: 保存済みかどうかの物理ディスクチェックおよびDB照合において、GIF / MP4 / 展開フォルダそれぞれの存在を正確に判定し、既に保存済みの作品は賢くスキップします。

---

### 4. 保存品質チェック (`verify_work_integrity`) の強化
- GIF・MP4動画が「0バイトファイル」になっていないかの検証。
- うごイラを1ファイルの動画（GIF/MP4）としてまとめた際、Pixiv側メタデータの総コマ数と比較されて「ページ数不足」と誤判定されないよう、専用の検査バイパスおよび正常性チェックを完備しています。

---

## ⚙️ 設定方法

1. `PixivVault.exe` を起動し、画面右上の **歯車アイコン（設定）** をクリックします。
2. 「**4. その他の設定**」項目内にある **「うごイラの保存形式」** ドロップダウンメニューを開きます。
3. 以下のいずれかを選択し、「**閉じる**」をクリックして設定を保存します。
   - `GIF動画 (デフォルト)`
   - `MP4動画`
   - `すべてフォルダ格納する (ZIP解凍+JSON)`

設定後ただちに、すべてのダウンロード機能において選択したフォーマットでうごイラが保存されるようになります。

## 🛠️ 技術仕様・ライブラリ情報
- **Python Version**: 3.13+
- **UI Framework**: Flet 0.85.3 (完全固定準拠)
- **Image/GIF Engine**: Pillow (PIL)
- **Video Engine (MP4)**: OpenCV (`cv2`) / imageio (`libx264` codec)
- **Build**: PyInstaller スタンドアロン実行ファイル (`PixivVault.exe`) 同梱済み
